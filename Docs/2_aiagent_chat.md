## Steps
- Click “Create Workflow.”

- Click “Add First Step.”

- Search for “Chat Trigger.”

- Click the “+” button to add it.

- Search for “AI Agent” and select it.

- Click the “+” button to add a chat model.

- Search for “Google Gemini Chat Model.”

- In the Credential or “Connect With” field, select “Create New Credential.”

- Open a new browser tab and search for “Gemini API.”

- Open the first link (Google AI / Gemini official site).

- Sign in to your Google account.

- Click “Get API Key.”

- Copy the API key.

- Go back to your Workflow.

- Paste the API key into the API Key field under Create New Credential.

- Click “Save.”

- Execute the Workflow.
